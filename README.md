# SkeltonDotNet6
C#アプリを作成する時にベースとなるプロジェクトファイル。.Net6版です<br>
<br>
プロジェクトテンプレートにして使います。<br>

* Json環境設定ファイルを自動保存。
* 多重起動禁止・フォームへのファイル・ディレクトリのD&D
